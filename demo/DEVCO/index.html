// 05 FAQ

FAQ

よくあるご質問をまとめました。

はい、可能です。週2〜3日の出社を推奨していますが、業務やチームの状況に応じて柔軟に運用しています。

書類選考 → カジュアル面談 → 技術面接 → 最終面接の流れです。応募から内定まではおおむね2〜3週間です。

ポテンシャル採用も行っています。基礎的な開発経験と学び続ける意欲があれば、職種未経験でもご応募いただけます。

入社後はオンボーディング期間を設け、メンターがつきます。技術書籍の購入やカンファレンス参加の支援制度もあります。

申請制で副業を認めています。本業に支障のない範囲であれば、スキルアップにつながる活動を歓迎しています。
